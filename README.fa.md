# Adobe Connect Local Recording Stitcher

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)
[![فارسی](https://img.shields.io/badge/Language-فارسی-green)](README.fa.md)

این راهنما به شما نشان می‌دهد چگونه یک ضبط (Recording) از Adobe Connect را از سرور دانشگاه دانلود کرده و با استفاده از یک اسکریپت محلی پایتون و FFmpeg آن را به یک فایل ویدیویی استاندارد MP4 تبدیل کنید.

### قدردانی و الهام‌بخشی

این پروژه تا حد زیادی از پروژه Vadana Class Downloader (VCD) الهام گرفته شده است. در حالی که VCD دانلود و رندر ویدیوهای Adobe Connect دانشگاه آزاد را به‌صورت آنلاین و از طریق شبکه انجام می‌دهد، این ابزار برای **پردازش آفلاین** طراحی شده است. این پروژه به‌طور ویژه سیستم تغییر یافته Adobe Connect دانشگاه فردوسی مشهد (Webinar) را هدف قرار می‌دهد و با استفاده از یک روش دانلود ZIP بومی برای دور زدن محدودیت‌های زمانی شبکه، داده‌های XML را به‌صورت محلی پردازش کرده و تایم‌لاین چندبخشی را با FFmpeg به یک ویدیو تبدیل می‌کند.

---

## مرحله ۱: دانلود فایل ZIP

1. شناسه ضبط (**Recording ID**) مربوط به Adobe Connect را پیدا کنید. برای مثال، پس از ورود به سیستم، در صفحه آرشیو کلاس‌ها به آدرس:

   https://pooya.um.ac.ir/educ/educfac/VClassRecords.php

   روی یکی از جلسات کلیک کنید. آدرس URL به شکل زیر خواهد بود:

   ```text
   https://webinar3.um.ac.ir/[RECORDING_ID]/
   ```

2. به‌صورت پیش‌فرض، سرور فایل ZIP را با نام Recording ID ذخیره می‌کند، اما می‌توانید نام فایل دانلودی را مستقیماً در لینک دانلود تغییر دهید. برای این کار، مقدار دوم `[RECORDING_ID]` را با نام دلخواه خود جایگزین کنید.

3. از ساختار لینک زیر استفاده کنید:

   ```text
   https://webinar3.um.ac.ir/[RECORDING_ID]/output/[DESIRED_NAME].zip?download=zip
   ```

> **مثال:** اگر شناسه شما `p1v8fvjnzey5` باشد و بخواهید فایل با نام `Session3` دانلود شود، لینک دانلود شما به شکل زیر خواهد بود:
>
> ```text
> https://webinar3.um.ac.ir/p1v8fvjnzey5/output/Session3.zip?download=zip
> ```

---

## مرحله ۲: استخراج فایل‌ها

پس از اتمام دانلود، محتوای فایل ZIP را در یک پوشه جدید روی رایانه خود استخراج (Extract) کنید.

*توجه: اسکریپت به‌صورت خودکار فایل MP4 نهایی را بر اساس نام همین پوشه نام‌گذاری می‌کند. برای مثال، اگر نام پوشه `Session3` باشد، خروجی نهایی `Session3.mp4` خواهد بود.*

---

## مرحله ۳: پیش‌نیازها

قبل از اجرای اسکریپت، مطمئن شوید موارد زیر روی سیستم شما نصب شده‌اند:

* **Python 3.x** نصب شده و به PATH سیستم اضافه شده باشد.
* **FFmpeg** نصب شده و به PATH سیستم اضافه شده باشد.
* *(اختیاری)* یک کارت گرافیک NVIDIA. اسکریپت به‌صورت پیش‌فرض تلاش می‌کند از CUDA (`h264_nvenc`) برای کدگذاری بسیار سریع ویدیو استفاده کند. در صورتی که کارت گرافیک NVIDIA ندارید، می‌توانید این قابلیت را در تنظیمات اسکریپت غیرفعال کنید.

---

## مرحله ۴: اجرای اسکریپت تبدیل

1. در پوشه استخراج‌شده (جایی که فایل‌های `.flv` و `.xml` قرار دارند)، یک فایل متنی جدید ایجاد کنید.

2. نام فایل را به `stitch.py` تغییر دهید (اطمینان حاصل کنید پسوند فایل از `.txt` به `.py` تغییر کرده باشد).

3. فایل `stitch.py` را با یک ویرایشگر متن باز کنید، کد اسکریپت پایتون موجود در مخزن GitHub را در آن قرار دهید و فایل را ذخیره کنید.

4. در همان پوشه‌ای که `stitch.py` قرار دارد، یک Terminal یا Command Prompt باز کنید.

5. اسکریپت را با دستور زیر اجرا کنید:

   ```bash
   python stitch.py
   ```

6. پس از پایان فرایند، فایل ویدیویی نهایی با زمان‌بندی صحیح و ترکیب‌شده، با نام پوشه (برای مثال `Session3.mp4`) در همان مسیر ایجاد خواهد شد.
